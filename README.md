# Chess Game

## Run
- Option 1: Open `index.html` directly in a browser
- Option 2: Start a local static server

```bash
cd /Users/yuanbo/Documents/chess-game
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls
- Click a piece, then click a highlighted square to move
- Supports castling, en passant, promotion, check/checkmate, and draws
- Click "Restart" to reset the game
- Use the opponent selector to play vs AI (AI plays black or white)
